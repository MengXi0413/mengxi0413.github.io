这是我的私人博客，记录了我的学习和日常生活：
1、考研日记：我将记录我每天的学习进度和状态，记录备战2027软微考研的过程
2、开发心得：我是集成电路科班出身，对数字芯片设计有着浓烈的兴趣，我将记录我学习芯片设计的心得和历程
3、日常琐事：生活总不是平淡无味的，我应该记录一些有意思，或幸福，或悲伤的琐事，留住一些终会被遗忘的时光

该博客使用github pages搭建在搭建该博客的时候，要做好内容板块的分区，并留好后续增加板块的接口。

当前实现与协作约定：

1、技术栈固定为 `Astro + GitHub Actions + GitHub Pages`
2、仓库名采用 `mengxi0413.github.io`
3、内容采用 `Markdown-first`，文章统一放在 `src/content/posts/`
4、栏目不按物理目录拆分，统一通过 frontmatter 的 `section` 字段区分
5、当前栏目固定为：
- `kaoyan`：考研日记
- `chip-design`：开发心得
- `life`：日常琐事
6、栏目元数据统一维护在 `src/config/sections.ts`
7、站点元数据统一维护在 `src/config/site.ts`
8、页面路由约定：
- 首页：`/`
- 单篇文章：`/posts/<slug>/`
- 栏目页：`/section/<section>/`
- 归档页：`/archive/`
- 标签页：`/tags/`
- 关于页：`/about/`
9、后台写作使用 `Pages CMS`，配置文件为仓库根目录的 `.pages.yml`
10、正文必须保持为标准 Markdown，不引入依赖服务端的动态能力
11、后续新增栏目时，优先扩展 `section` 配置，而不是打散现有信息架构
12、视觉风格保持“温暖纸张感、清晰分区、轻手帐感”，避免默认模板味道过重
